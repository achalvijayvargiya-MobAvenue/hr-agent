# HR Agent - Product Requirements Document

## Goal
Match candidate CVs to Job Descriptions (JDs) and rank candidates by relevance.

## Primary Use Cases
1. Upload JD PDFs
2. Upload CV PDFs
3. Extract structured information
4. Match candidates to jobs
5. Rank candidates
6. Provide explainable scoring

## Success Metrics
- Precision@10
- Recall@50
- Recruiter acceptance rate
- Processing latency
- Cost per match

## Non Goals V1
- Interview scheduling
- Candidate outreach
- ATS integration
